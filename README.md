# Practice Code

This repository contains various programming exercises, university lab work, and competitive programming solutions.

- `C_programming/`: A collection of C programming practice, including lab work and exercises.

- `Python/`: Basic Python scripts and experiments.
- `HTML Docs/`: University-related documents such as routines.
